# Family Log System

Purpose: durable long-term logs for family life, health, habits, incidents, and important decisions.

## Rules

- Use dated entries.
- Keep logs factual, short, and searchable.
- Use categories/tags in every entry.
- Use Asana only for action items; use these files for history.

## Files

- `roshan.md` — Roshan personal health/life log
- `nadee.md` — Nadee personal health/life log
- `leon.md` — Leon log
- `maylee.md` — Maylee log
- `family-decisions.md` — major family decisions, plans, commitments
- `incidents.md` — notable issues, disruptions, accidents, conflicts, setbacks

## Entry format

```markdown
## YYYY-MM-DD
- type: health | food | habit | sleep | routine | decision | incident | admin | school | work | family | appointment | medication | symptom | idea
- tags: [tag1, tag2, tag3]
- summary: one-line summary
- details: short factual notes
- follow-up: optional next step
```

## Search guidance

- Search by person file first.
- Search `family-decisions.md` for decisions/agreements/plans.
- Search `incidents.md` for problems/issues/events.
- Use tags like `health`, `eating`, `sleep`, `doctor`, `school`, `behavior`, `medication`, `wife`, `kid`, `family`, `decision`.
